	3，设置babiesCommander作为全局掌控者，不断观看地面以显示地面情况，向HuLuBaby发送讯息，命令其移动到指定位置或报告自身属性，其大脑可以随机出数字序列从而命令葫芦娃们站成随机地一排从而实现排序和报数的模拟
